# ECU-Based Telemetry Dashboard

## Quickstart

First, install dependencies with:

    $ npm install

Then, run the proyect using *vite's* dev server:

    $ vite

## Directory

The proyect is structured like this:

```diff
+ ├─┬ electron
+ │ ├─┬ main
+ │ │ └── index.ts    entry of Electron-Main
+ │ └─┬ preload
+ │   └── index.ts    entry of Preload-Scripts
  ├─┬ src
  │ └── main.ts       entry of Electron-Renderer
  ├── index.html
  ├── package.json
  └── vite.config.ts
```
